# Tattarrattat

## An ASP.NET C# application to that checks if a string is a palindrome.

    - CSS SASS Bootstrap Layout
    - ASP.NET MVC
    - Entity Framework
    - C# Fundamentals
    - C# Loops
    - C# Functions
    - C# Dom Manipulation
    - C# If/Then/Else
    - C# Boolean Logic
